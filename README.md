# TypeScript Library Template

> Production-ready **TypeScript library template**.

## Features

- ESM + CJS dual package support
- Powered by **tsdown**
- Vitest + Coverage (V8)
- Biome (lint) + Prettier (format)
- Husky + lint-staged pre-commit setup
- Proper `exports` field configuration
- CLI ready
- Clean `dist` output
- Node 18+ support
- Optimized for publishing to npm
- Enterprise-grade GitHub Actions CI with parallel jobs and PR governance

## Getting Started

### Use This Template

```bash
git clone https://github.com/teneplaysofficial/ts-lib-template.git my-lib
cd my-lib
pnpm install
```

Or click **Use this template** on GitHub.

## Scripts

| Script                | Description                      |
| --------------------- | -------------------------------- |
| `pnpm build`          | Build library with tsdown        |
| `pnpm test`           | Run tests with coverage          |
| `pnpm lint`           | Check code with Biome            |
| `pnpm lint:fix`       | Fix lint issues                  |
| `pnpm fmt`            | Format using Prettier            |
| `pnpm fmt:check`      | Check formatting                 |
| `pnpm typecheck`      | TypeScript check                 |
| `pnpm prepublishOnly` | Runs test + build before publish |
